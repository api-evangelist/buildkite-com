The Buildkite MCP Server can now do more than read from Buildkite. It can take action across clusters, builds, jobs, and pipeline schedules. New cluster and queue management tools let agents list, inspect, create, and update Buildkite clusters and cluster queues, and pause or resume queue dispatch.
